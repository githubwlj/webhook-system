📚 API文档

完整的Webhook系统API文档和使用指南

🔗 基础信息

服务器地址: https://webhook.itdb.top

API版本: v1.0

认证方式: Bearer Token

📡 接收Webhook

POST请求到以下地址接收webhook数据：

https://webhook.itdb.top/webhook/receive/{WEBHOOK_ID}

📤 发送Webhook

POST请求到以下地址发送webhook：

POST https://webhook.itdb.top/webhook/send
Content-Type: application/json
Authorization: Bearer YOUR_TOKEN

{
  "url": "https://example.com/webhook",
  "payload": {
    "message": "Hello World",
    "timestamp": "2024-01-01T00:00:00Z"
  },
  "secret": "your_secret_key"
}

🔐 身份验证

系统支持多种令牌类型：

  • 管理员令牌: 完全访问权限
  • UI访问令牌: 仅可访问监控界面
  • API访问令牌: 仅可调用API接口
  • Webhook密钥: 用于签名验证

🛡️ 安全特性

  • 请求频率限制
  • 签名验证
  • 访问日志记录
  • IP白名单（可选）

📊 监控统计

系统提供完整的监控数据：

  • 请求成功率统计
  • 响应时间监控
  • 错误日志分析
  • 实时活动监控
